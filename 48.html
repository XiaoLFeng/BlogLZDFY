插件介绍

本插件适用于大厅使用，如果你要放入生存服等服务器也不是不行，如果出现问题请自行查清楚，如果遇到插件本身问题，请留言。
HubHats给玩家一个GUI操作界面的帽子管理插件，适用于大厅，游戏服那种给予VIP后解锁帽子使用的插件。
每一个帽子都有一个权限，所以你可以分很多个VIP来分配不同的权限 （就算白痴也能用，就一个GUI←我指的是玩家）

指令：

  • /hubhats - 基本插件，然而只会显示作者信息
  • /hubhats help - 显示插件帮助，但是是英文（作者声明不打算开源，故无法进行核心汉化）
  • /hubhats reload - 重载配置（重载config.yml）
  • /hubhats setjoinitem - 更改入服给予的ITEM（物品）

/hubhats 使用

我不进行详细说明，因为这条指令真的没什么用，入下图所示

/hubhats help 使用

用处不大，我上面指令已经写出来了，如下图所示

/hubhats reload 使用

重载插件，此操作只会重载config.yml信息，如果你想修改message.yml或者hat.yml文件，请重启哦 不然的话输入“/hubhats reload”操作是没有用处的哦

/hubhats setjoinitem 使用

这条指令是设置给予玩家打开GUI使用的指令，就是替换给予的物品，如下图Gif表示（图有40mb流量党请注意省流操作） 其实就是美观用的，不过不支持附属ID（我忘记怎么表达了，反正就是例如【4:1】后面的1是不能表示的。【如果配置文件修改后可以，请在此纠正】）

权限环节

本插件权限（其实是帽子允许佩戴权限） hubhats为基本权限，后面的英文可以到hat.yml进行查看对应名字就是对应权限
使用gui权限 hubhats.use
使用reload权限 hubhats.reload
