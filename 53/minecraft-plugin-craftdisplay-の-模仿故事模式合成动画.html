[title]插件授权[/title] [title]前提说明[/title] 如果您使用的是Spigot 1.13至1.13.2 请使用CraftDisplay 5.3.0。都需要权限组 如果您使用的是Spigot 1.12.2或更低版本+GM权限组 请使用CraftDisplay 5.2.0。不需要权限组 如果您使用的是Spigot 1.12.2或更低版本+LP权限组 请使用CraftDisplay 5.2.0。需要权限组 请注意了！！！在1.7.10的 KCauldron 是无法正常工作的，望周知（插件无法在1.7.10运行，1.7.10开服玩家“安息吧”） 在此感谢MCBBS的  水杉 如果您版本更高，请查看帖子（支持1.10~1.15）：https://www.mcbbs.net/thread-1010187-1-1.html [title]bilibili帮助[/title] [bilibili cid=”72718024” page=”1”]41403859[/bilibili]   教程 [bilibili cid=”74282461” page=”1”]42321893[/bilibili] [title]基本介绍[/title] 其实也不用什么介绍，因为没有什么插件的说明，就是为了服务器好看 增加服务器的逼格，你可以随时开启关闭服务器这些特效显示 [title]指令及权限[/title] 指令帮助

  • /craftdisplay crafting_table [ enable / disable / toggle ]     开启或关闭 CraftDisplay （工作台）
  • /craftdisplay chest [ enable / disable / toggle ]     开启或关闭 ChestDisplay （箱子）
  • /craftdisplay furnace [ enable / disable / toggle ]     开启或关闭 FurnaceDisplay （熔炉）
  • /craftdisplay enchanting_table [ enable / disable / toggle ]     开启或关闭 EnchantDisplay（末影箱）
  • /craftdisplay anvil [ enable / disable / toggle ]     开启或关闭 AnvilDisplay （不知道，铁砧吧）
  • /craftdisplay reload      重载插件
