基于UptimeRobot的云端状态监控系统
发表于2020-04-12|更新于2020-08-12|闲聊实用
|阅读量:|评论数:

[dangerbox title=”⚠前言”] 此网站已有开源，我会将此网站进行二次开发 请支持原作者GitHub 在更新后我会将我的GitHub进行发布，不过我还是建议你下载原作者的，尊重原作者 [/dangerbox]

[title]网址[/title] 状态监控网站（主用）：https://check.xiaolfeng.xyz/ 状态监控网站（备用）：https://www.xiaolfeng.xyz/expand/Status/ 注：此网站供访客和我进行查看，如果此两个网站某原因无法打开请不要着急，我还有一个监控网站挂在我自家服务器里面，不用担心无法进行问题监控   [title]用处[/title] 没有什么实际用处，主要是给我看的，或者（将来）我有开发API或者开发文档存储后，作为开发需要 有其他用户进行下载，下载无法连接情况即可查询此站。   [title]介绍[/title] 基于 Upscuits，以 UptimeRobot 为数据源的在线状态统计页面。 2016-10-06 Uptime Robot 已经新增 Public Status Pages 功能，详见 官网。 2019-02-15 Cup 已支持 Uptime Robot APIv2，请尽快更新您的 Cup 版本，Uptime Robot 已停止支持 APIv1。

使用方法

注册 UptimeRobot，获取 Monitor-Specific / Read-Only API Key(s) 将 API Keys 填入 js/config.js 修改 index.html 内的页面标题等信息 完成！   [title]具体使用方法[/title] 进入上方给的网站，注册账号后添加 左上角Add添加站点 此处可以选择用Http/KeyWord/Ping/Port模式进行连接（Pro版本不详细说明，我也不知道，没钱买） 然后点击上面的My Settings进行API分发 找到 API Settings 中间点 Show/hide It 然后进行输入你刚才的设定名字，创建 就会分得一个m开头的一串符，就是API，复制API打开config.js 替换掉那一串API即可 然后打开你的index.html就可以看到你的服务器状态了~ （PS：更新需要5分钟，所以有可能没更新出来显示红色X）
